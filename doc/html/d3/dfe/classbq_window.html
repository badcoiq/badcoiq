bqGUIWindow * bqWindow::CreateNewGUIWindow ( const bqVec2f & position,
const bqVec2f & size )

Добавить GUI окно.

GUI окна рисуются в системные окна. При создании GUI окна, установка происходит автоматически. Рисование, обновление и перестройка GUI происходит не по общему списку GUI окон как это было ранее, а в соответствии с системными окнами. Далее должно быть проще реализовать рисование во множество системных окон.

◆ RebuildGUI()

void bqWindow::RebuildGUI ( )

Обновить расположение всех окон и элементов внутри.

Вычисляются все области (для рисования, для определения нажатия, clip rect и прочие), вычисляются позиции например текста. Так-же прочие вычисления, когда достаточно обновить только 1 раз при изменении какого-то значения. Например, изменили размер окна - вызвали RebuildGUI(); Добавили или убрали элемент - вызвали RebuildGUI();
