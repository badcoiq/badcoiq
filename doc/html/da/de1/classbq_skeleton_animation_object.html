Простая инициализация на основе скелета и анимации. В анимации берётся имя джоинта, потом происходит поиск джоинта в скелете (поиск по имени). Если найдено то будет добавлен джоинт. По сути нужен более полный доступ для гибкой настройки, но пока будет так.
